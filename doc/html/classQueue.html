Descripción detallada

template<typename T = int>
class Queue< T >

Esta clase template ofrece una implementación de cola con operaciones para agregar, eliminar elementos, y consultar el tope de la cola, entre otras.

Parámetros de plantilla
TTipo de elementos almacenados en la cola.
